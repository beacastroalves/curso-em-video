Alinhamento de textos usando CSS

Usando o text-align


Este primeiro parágrafo terá o alinhamento padrão, à esquerda.


O segundo parágrafo ficará alinhado à direita.


O terceiro parágrafo terá alinhamento centralizado.


O último parágrafo terá o alinhamento justificado. Para que você perceba isso, vou colocar um texto um pouco maior e uma margem de 16px nas laterais. Caso você não perceba que o alinhamento está sendo feito nas duas laterais, tenta diminuir um pouco o tamanho do seu navegador. Esse tipo de alinhamento é bastante utilizado para textos longos e grandes parágrafos em documentos como TCC, para dar uma sensação de simetria.
